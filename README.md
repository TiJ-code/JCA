# JCA

# Java Chat Application

A privacy-focused, self-hostable open-source messenger written in Java.

---

## About

JCA will be a communication platform designed around:
- User ownership.
- Privacy.
- Self-hosting.
- Extensibility.

The goal is to create a messenger where individuals and communities can control their own communication infrastructure.

---

## Vision

Modern communication platforms are convenient, but users usually do not control:

- Where their data is stored.
- How their communication is handled.
- Which services they depend on.

JCA aims to provide an alternative:

- Run your own server.
- Own your data.
- Use open-source clients.
- Keep communication private.

---
